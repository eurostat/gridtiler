# GridTiler

Produce tiled grids in [tiled grid format](https://github.com/eurostat/gridviz/blob/master/docs/tiledformat.md) for web mapping applications such as [GridViz](https://github.com/eurostat/gridviz).



Grid tiling in: 
- [Python](/py)
- [Java](/java)
